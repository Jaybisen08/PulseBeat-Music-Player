## 🌟 Overview

PulseBeat is a beautifully designed web-based music player that combines a modern interface with essential playback features. It offers a smooth and immersive user experience with animated transitions, responsive layouts, playlist support, and customizable themes.

Whether you're listening to your favorite songs or exploring the interface, PulseBeat delivers a clean and engaging music experience.

---

## ✨ Features

- 🎵 Play & Pause Music
- ⏮ Previous / Next Track
- 📂 Playlist Support
- 🎼 Song Title & Artist Display
- ⏱ Live Progress Bar
- 🔊 Volume Control
- 🌙 Dark / Light Mode
- 💾 Saves Theme & Volume using Local Storage
- 📱 Fully Responsive Design
- 🎨 Modern Glassmorphism UI
- ⚡ Smooth GSAP Animations
- 🎧 Mini Music Player
- 🔄 Autoplay Next Song
- ❤️ Beautiful Album Artwork

## 🚀 Getting Started

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/Jaybisen08/PulseBeat-Music-Player.git
```

### 2️⃣ Open the Project

```bash
cd PulseBeat-Music-Player
```

### 3️⃣ Run the Project

Simply open **index.html** in your browser.

No additional installation is required.
